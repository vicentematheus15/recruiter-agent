# Recomendações de Cursos Alura

| Curso | Breve descrição | Link |
|---|---|---|
| Fundamentos de Python para Data Science | Introdução ao Python, bibliotecas essenciais (NumPy, Pandas) e manipulação de dados – ideal para quem vem de JavaScript. | https://cursos.alura.com.br/course/fundamentos-python-data-science |
| Estatística Básica para Análise de Dados | Conceitos estatísticos fundamentais (tendência, dispersão, testes de hipóteses) aplicados a dados reais. | https://cursos.alura.com.br/course/estatistica-basica-analise-dados |
| Introdução ao Machine Learning com Scikit‑Learn | Criação de modelos preditivos simples (regressão, classificação) usando Scikit‑Learn, com foco em interpretação. | https://cursos.alura.com.br/course/machine-learning-scikit-learn |
| SQL Avançado para Análise de Dados | Consultas avançadas, sub‑queries, CTEs, funções de janela e otimização em PostgreSQL/MySQL. | https://cursos.alura.com.br/course/sql-avancado-analise-dados |
| Data Visualization com Python (Matplotlib & Seaborn) | Criação de gráficos estáticos e interativos para comunicar insights de forma clara. | https://cursos.alura.com.br/course/data-visualization-python |
| Big Data com Apache Spark (PySpark) | Processamento distribuído de grandes volumes de dados usando Spark, com exemplos práticos de ETL. | https://cursos.alura.com.br/course/big-data-apache-spark-pyspark |
| Introdução ao Power BI | Construção de dashboards interativos, conexão a múltiplas fontes e storytelling visual. | https://cursos.alura.com.br/course/power-bi-introducao |
| Data Storytelling: Comunicação de Resultados | Transforme análises técnicas em narrativas compreensíveis para stakeholders não técnicos, reforçando sua soft skill de comunicação. | https://cursos.alura.com.br/course/data-storytelling-comunicacao-resultados |
